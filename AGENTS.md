# Agent Instructions

TypeScript/Node.js authentication system implementing OAuth/OIDC endpoints using Hono, MongoDB/Mongoose, and Zod.

## Quick Reference

- **Package Manager**: npm
- **Build**: `npm run build` (uses `tsc && tsc-alias` - note the alias resolution step)
- **Typecheck**: `npm run typecheck` (tsc --noEmit)
- **Test Framework**: Vitest (`npm run test`)
